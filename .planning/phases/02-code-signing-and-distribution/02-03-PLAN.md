```bash

  --private \
```

Complete the following in order. These require your Apple Developer account and App Store Connect access — they cannot be automated.

A. Set missing GitHub Secrets (if Task 1 found any absent):

For APPSTORE_KEY_ID, APPSTORE_PRIVATE_KEY, APPSTORE_ISSUER_ID:
1. Go to https://appstoreconnect.apple.com/access/integrations/api
2. Create an API key with role "App Manager" (minimum for TestFlight uploads)
3. Download the .p8 file — it can only be downloaded once
4. Copy the Key ID and Issuer ID shown on that page
5. Run:
   ```bash
   gh secret set APPSTORE_KEY_ID --body "YOUR_KEY_ID" --repo IgorGanapolsky/openclaw-console --env production
   gh secret set APPSTORE_ISSUER_ID --body "YOUR_ISSUER_ID" --repo IgorGanapolsky/openclaw-console --env production
   gh secret set APPSTORE_PRIVATE_KEY --body "$(cat /path/to/AuthKey_*.p8)" --repo IgorGanapolsky/openclaw-console --env production
   ```

For MATCH_PASSWORD (choose a strong password to encrypt certs):
```bash
gh secret set MATCH_PASSWORD --body "YOUR_MATCH_PASSWORD" --repo IgorGanapolsky/openclaw-console --env production
```

For MATCH_GIT_BASIC_AUTHORIZATION (base64 of `x-access-token:YOUR_GITHUB_PAT`):
```bash
ENCODED=$(echo -n "x-access-token:YOUR_GITHUB_PAT" | base64)
gh secret set MATCH_GIT_BASIC_AUTHORIZATION --body "$ENCODED" --repo IgorGanapolsky/openclaw-console --env production
```

For ADMIN_TOKEN (GitHub PAT with repo scope — may already exist):
```bash
gh secret set ADMIN_TOKEN --body "YOUR_GITHUB_PAT" --repo IgorGanapolsky/openclaw-console --env production
```
