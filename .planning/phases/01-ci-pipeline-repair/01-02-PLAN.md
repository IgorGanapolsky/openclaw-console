       `versionCode = (System.getenv("GITHUB_RUN_NUMBER") ?: "1").toInt()`

    5. CHANGE the Compose BOM line:
       FROM: `val composeBom = platform("androidx.compose:compose-bom:2024.01.00")`

    6. REMOVE the entire `composeOptions` block:
       ```kotlin
       composeOptions {
           kotlinCompilerExtensionVersion = "1.5.4"
       }
       ```
       Reason: With Kotlin 2.x and the `org.jetbrains.kotlin.plugin.compose` plugin declared in build.gradle.kts,
       `kotlinCompilerExtensionVersion` is no longer needed — the plugin version matches the Kotlin version automatically.
       Keeping it causes a build warning or error with AGP 8.7 + Kotlin 2.1.

    Do NOT change any dependency versions other than the Compose BOM. Do NOT change minSdk (leave at 28 — this is a known discrepancy from CLAUDE.md spec of 26 but is out of scope for this CI-03 task).
