# OpenClaw Console Marketing

## Structure

- `data/` — Live metrics, growth snapshots, campaign data
- `diagrams/` — Marketing diagrams and visuals
- `keywords/` — ASO keyword strategy and rankings
- `posts/` — Social media and blog content
- `compliance/` — Store review responses, policy evidence
- `growth-evidence/` — Conversion funnels, A/B tests, DAA tracking
- `product-pages/` — Custom store listing variants
- `store_growth/` — ASO strategies, competitor analysis
