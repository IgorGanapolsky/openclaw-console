# Broward Gym Outreach — 2-Week Sprint

## Goal
Get 3 to 5 coaches to run one class drill using Random Tactical Timer and provide feedback.

## Outreach Rules
- Ask for feedback, not promotion.
- Keep message under 4 lines.
- Follow up once after 72 hours.

## Initial Message
"Hey coach — I built a free random reaction timer for boxing/BJJ/combatives drills. If you want, I can share 2 quick drill presets your class can test this week. I'd value honest feedback."

## Follow-Up Message
"Quick bump — if useful, I can send a one-page drill card (boxing + BJJ preset ranges) so you can test it in one round block."

## Drill Presets
- Boxing flurry interrupt: 30s to 90s random trigger, 10s alarm.
- BJJ scramble spike: 20s to 120s random trigger, 15s alarm.
- Conditioning chaos rounds: 15s to 75s random trigger, loop enabled.

## Capture Checklist
- Coach name + gym
- Drill used
- What improved / what failed
- Quote permission (yes/no)

## Output
At end of week 2, publish one summary post: "What 5 gyms said after testing random reaction cues."
